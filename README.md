# 知识库服务
该服务负责文档管理，用户可以对文档进行一系列的操作并提供更好的用户体验。

## 特点
- **文档管理：可以快速操作文档，并支持文档上传附件、评论、记录操作日志等功能**
- **版本管理：记录用户每次保存文档的版本记录，并支持版本回滚、比较、删除操作**
- **操作简单快速**
